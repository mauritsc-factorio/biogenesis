- **Science:** SEA (Systems Emergence Analysis) production established

**What changes:** Everything the player has built — every system, every organism, every technology — converges into a single project: the Arcology. The player's final act is not to build something. It's to let go.

## 2. THE THREE ACTS OF ERA 9

Era 9 is structured differently from previous eras. Instead of a linear timeline of unlocks, it has three OVERLAPPING acts that the player pursues simultaneously:

### Act I: Chimera Commissioning (hours 0-15)

The arcology needs 12 chimera organisms (designated XB-07 through XR-00). Each is a multi-step project:

```
1. DESIGN: Genome Authoring Platform → chimera genome (chassis + 3-4 gene cassettes + promoter)
2. EVOLVE: Evolution Engine → optimize for target fitness (30-45 min per run)
3. VALIDATE: Phenotypic Validation Chamber → confirm phenotype
4. CULTURE: Scale production in bioreactors
5. DEPLOY: Install in target arcology module
6. INTEGRATE: Monitor Bio Web interactions with existing chimeras. Adjust if conflicts arise.
```

Each chimera takes 2-4 hours of real time (including evolution runs, validation attempts, and scaling). Some fail validation (30-45% rejection for 3-4 cassette organisms). Some interact badly with chimeras already deployed — the player must redesign or swap deployment locations.

### Act II: Ecological Tuning (hours 8-25)

Once 4+ chimeras are online, the arcology's ecology begins to LIVE. Nutrient flows, population dynamics, metabolic cycles — they all oscillate, overshoot, and crash before stabilizing. The player's job: diagnose and treat.

**The Biosphere Equilibrium Index (BEI).** A new metric displayed on the Living System Integrity Dashboard. BEI ranges from 0 (chaos) to 100 (perfect equilibrium). The arcology becomes self-sustaining at BEI ≥75.

| BEI Range | State | What's Happening |
|-----------|-------|------------------|
| 0-25 | Chaos | Populations crash and spike wildly. Nutrient flows unpredictable. Waste accumulates. Early commissioning state. |
| 25-50 | Oscillating | Boom-bust cycles. Nutrient peaks and valleys. The system is trying to find equilibrium but overshooting. |
| 50-75 | Stabilizing | Oscillations dampening. Occasional spikes. The player's adjustments are taking hold. |
| 75-90 | Equilibrium | The arcology is self-sustaining. Nutrient flows stable. Populations balanced. Minimal intervention needed. |
| 90-100 | Emergent | The ecology is doing things the player didn't design. New relationships forming. Organisms adapting to each other. The biosphere is becoming AUTONOMOUS. |

**Tuning tools:**
- **Nutrient Flow Regulator:** Adjust vascular conduit flow rates to specific modules (increase/decrease nutrient delivery)
- **Population Ratio Controller:** Set target ratios for chimera populations within modules (more of one, less of another)
- **Metabolic Dampener:** Applied to overproducing organisms to reduce output temporarily
- **Ecological Stimulus:** Applied to underperforming organisms to boost output temporarily
- **Emergency Flush:** Purge a module's waste buildup (uses large amount of purified water, resets waste level)

Each adjustment propagates through the ecosystem with **delayed consequences** (5-15 min). Increasing nutrients to the Biodome makes crops grow faster → more CO₂ produced → Atmospheric Processor works harder → more waste → Recycler stressed → nutrient quality drops in other modules. The player learns to anticipate cascades.

**Ecological Events (keeping tuning ACTIVE, not passive).** Random events fire every 30-60 min during Act II, requiring immediate player response:

| Event | Trigger | Consequence | Response |
|-------|---------|-------------|----------|
| Population Bloom | BEI 30-50, random chimera overproduces | One chimera's population spikes 200%. Crowds out neighbors. BEI drops 5-10 points. | Apply Metabolic Dampener to blooming chimera. Or: redirect nutrients away from affected module. |
| Nutrient Crash | BEI 50-75, nutrient supply disruption | One module's nutrient level plummets. All organisms in module slow 50%. Cascade risk to connected modules. | Emergency Flush + redirect nutrient flow. Temporary — stabilizes in 10 min if handled quickly. |
| Chimera Drift | BEI 60-80, chimera begins expressing unexpected traits | A chimera's behavior shifts. Previously mutualistic relationship becomes neutral. Production drops in affected chain. | Re-evolve the chimera (short 15-min run with existing culture) or adjust module composition. |
| Immune Overreaction | BEI 70-90, XB-71 targets a benign chimera | Immune Sentinel mistakes a productive chimera for contamination. Attacks it. Module output drops 30%. | Recalibrate XB-71 via Neural Integrator (if deployed) or temporarily disable immune response in affected module. |
| Spontaneous Symbiosis | BEI 80+, two chimeras form an undesigned bond | NEW mutualistic relationship appears. BEI jumps +3-5. A pleasant surprise. | No action needed — observe and enjoy. The ecology is doing something you didn't plan. This is the sign of a healthy biosphere. |

Events keep the player ENGAGED during the tuning phase. They're not watching a number go up — they're diagnosing and treating a living system that keeps surprising them. The ratio of problems to pleasant surprises shifts as BEI rises: below 50, mostly problems; above 80, mostly spontaneous symbioses.

**BEI progression rate:**
- Each deployed chimera adds +5-8 BEI instantly (depending on interactions with existing chimeras)
- Passive equilibrium seeking: +1 BEI per hour (slow without intervention)
- Successful ecological event response: +5-8 BEI per event (the primary driver)
- Failed ecological event response: -3-5 BEI per event
- Events fire every 20-40 min (more frequent than the 30-60 in early design — active tuning should feel ACTIVE)
- **Projected timeline:** 12 chimeras deployed (+60-90 BEI) → BEI ~70 from commissioning alone → tuning from 70 to 95 requires ~5-8 successful event responses over ~4-6 hours

**The satisfaction:** BEI climbing from 30 to 75 as chimeras deploy is steady and rewarding. The push from 75 to 95 through active ecological tuning is the endgame's core challenge — 4-6 hours of engaged problem-solving, not 17 hours of monitoring. When BEI hits 75 and the Dashboard turns blue ("Self-Sustaining"), it's the most earned achievement in the game.

**The Autonomous Variant (Era 9 specimen).** This is the final breadcrumb — and it only appears once the arcology's BEI reaches ≥90 (Emergent state). At BEI 90, the ecology begins producing organisms WITHOUT player input. The Autonomous Variant is the first: an organism the biosphere created on its own. It appears as an unrecognized entity in the Ecology Monitor module.

*"Organism detected: unknown classification. This entity was not designed, not evolved in your engines, not fused in your chambers. It emerged from the arcology's ecology — produced by the interactions between your chimeras. The first life not descended from your designs. The biosphere is self-generating."*

The player collects the Autonomous Variant. Now they have all 9 (plus the extra Era 1 and Era 3 specimens — 11 total, but XR-00 needs 9 specific ones, one per era. The duplicates are insurance).

**The Omega Organism design:**
```
Genesis Transcription Array (4×4, 50 kW — the final signature machine):
  Input: 9 Archived Specimens (one per era) + Omega Genome Blueprint + Omega Gene Cassette Set
  → 48 min growth cycle (the longest single recipe in the game)
  → Output: Omega Organism XR-00

Where the inputs come from:
  Omega Genome Blueprint: Genome Authoring Platform
    → Blank Chassis (special: Universal Chassis, crafted from ALL 4 chassis types combined)
    + Gene cassettes incorporating data from each archived specimen

```

Expensive, slow, but possible. The player reconstructs the specimen's genome from partial data scattered across their organism library. This is a SAFETY NET — no 150-hour playthrough should end because of a 2% drop chance at hour 3. Players who collected all specimens naturally skip this entirely.

**The Omega Organism doesn't DO one thing.** It connects ALL chimeras into a unified ecosystem. When deployed, XR-00 establishes Bio Web relationships with EVERY chimera in the arcology. It's the keystone species — the organism that makes the whole biosphere work as one. Without it, the chimeras cooperate but remain separate entities. With it, they form a UNIFIED LIVING SYSTEM.

## 4. ARCOLOGY MODULES (detailed)

Each module is a living architecture zone with specific chimeras, functions, and construction requirements. Era 8 built prototypes of Biodome and Workshop. Era 9 scales these and adds 5 more.

### Module Construction

Each module grows from a **Module Architecture Seed** — a large-scale living architecture seed that defines the entire module structure.

```
Module Architecture Seed recipe (per module type):
  Living Architecture Seed Incubator (120s):
  1 Module Blueprint (specific to type) + 4 Self-Healing Wall Seed + 2 Ceiling Seed + 2 Floor Seed
  + 4 Vascular Conduit Segment + 2 Neural Mesh Node + 1 Nutrient Distribution Hub + 1 Lymphatic Filter Node
  + Module-specific components (see each module below)
  → Module Architecture Seed

Deployment: place seed on 12×12 cleared area → 48 min growth cycle (continuous nutrient solution feed required)
  → Module grows: walls, ceiling, floor, internal conduits, mesh, filtration — ALL from one seed
  → Post-growth: deploy chimeras, configure systems, connect to other modules via conduits
```

### The Seven Modules

**1. Biodome** (crop production, atmosphere)
- Size: 12×12. Special components: 2 Photosynthetic Membrane + 1 Growth Accelerator
- Chimeras: XB-66 (Photosynthetic Canopy) + XB-91 (Pollination)
- Function: enclosed crop production at maximum efficiency. Atmospheric cycling. Pollination maintains genetic diversity.
- Output: all crop products at +60% yield (stacking all bonuses: biodome +40%, mesh +5%, canopy +15%)

**2. Aquatic Systems** (water processing, fish)
- Size: 12×12. Special components: 2 Bio-Membrane + 1 Aquatic Ring Framework
- Chimeras: XB-58 (Aquatic Filtration)
- Function: closed-loop water processing. ALL water recycled. Fish production integrated.
- Output: zero water waste. Fish yield +40%.

**3. Energy Nexus** (power distribution)
- Size: 12×12. Special components: 4 Bioelectric Harvester + 2 Symbiotic Conduit
- Chimeras: XB-77 (Energy Converter)
- Function: arcology-internal power. Converts chimera metabolic activity → electricity. Distributed to all modules.
- Output: Arcology Core power (self-sustaining — no fuel input once equilibrium reached)

**4. Workshop** (fabrication)
- Size: 12×12. Special components: 2 Fabrication Matrix + 1 Spider Silk Panel
- Chimeras: XB-31 (Structural Secretor)
- Function: biological fabrication. The Secretor chimera produces structural materials on demand.
- Output: all machines within module: +15% productivity, +10% speed (mesh + matrix + secretor assistance)

**5. Research Wing** (science, Genesis Array)
- Size: 12×12. Special components: 2 Fused Silica + 1 Genome Authoring Platform (integrated)
- Chimeras: XB-99 (Neural Integrator)
- Function: houses the Genesis Transcription Array. The Neural Integrator chimera coordinates all arcology-wide neural mesh.
- Output: Genesis Transcription Array operational. Final science pack production.

**6. Habitat** (living quarters)
- Size: 12×12. Special components: 2 Atmospheric Processor Culture + 1 Thermal Insulation Layer
- Chimeras: XB-07 (Thermal Regulator) + XB-19 (Atmospheric Processor)
- Function: player living space. Maintained atmosphere. Comfortable temperature. Aesthetic.
- Output: no production function. Thematic — the player LIVES in the arcology.

**7. Ecology Monitor** (biosphere tracking)
- Size: 12×12. Special components: 4 Living System Dashboard + 2 Neural Mesh Node (upgraded)
- Chimeras: XB-42 (Soil Architect) + XB-71 (Immune Sentinel) + XB-85 (Nutrient Recycler)
- Function: monitors BEI. Contains intervention tools. Processes waste. The arcology's immune system and digestive system in one module.
- Output: Biosphere Equilibrium Index tracking. Ecological intervention capability.

**Power: Arcology Core (self-sustaining)**

The Arcology Core doesn't produce a fixed MW number. Instead, it harvests energy from the arcology's internal metabolic activity — the sum of all chimera metabolism, all living architecture bioelectricity, all vascular conduit pumping. When BEI ≥75, the arcology produces MORE energy than it consumes internally. The surplus is available for the player's external factory.

At BEI 75: arcology is energy-neutral (internal systems self-powered, no surplus)
At BEI 90: arcology produces ~5 MW surplus
At BEI 100: arcology produces ~15 MW surplus (equivalent to Metabolism Reactor — but from the building itself)

The player's external factory still uses inherited power (Metabolism Reactor, Enzymatic Turbine, etc.). The Arcology Core supplements, not replaces. Eventually (post-endgame), the player could build additional arcology modules and generate all power from living architecture.

When XR-00 is deployed, it creates Bio Web relationships with EVERY chimera. All competitive relationships become neutral. All mutualistic relationships gain +10%. All commensal relationships become mutualistic. XR-00 is the universal connector — it makes the ecosystem cooperate.

## 10. BALANCE ANALYSIS

### Chimera Commissioning Rate

**12 chimeras needed.** Each requires: design (5 min crafting) + evolution (30-45 min) + validation (~35% rejection = ~1.5 attempts average) + culture scaling (30 min) + deployment (10 min).
**Per chimera: ~2-3 hours including failures and retries.**
**With 2 Evolution Engines staggered: ~1 chimera per 1.5 hours.**
**12 chimeras: ~18 hours of commissioning work.** Overlaps with Acts II and III — the player commissions chimeras while tuning the growing ecology.

### BEI Progression

BEI starts at 0 when the first chimera is deployed. Each chimera adds ~5-8 BEI points (depending on interactions). With 6 chimeras: BEI ~30-50. With 12: BEI ~60-90 (before tuning).

**Tuning raises BEI from ~70 to 95+ over 4-6 hours of active ecological event management.** Chimera deployment alone gets BEI to ~70. Ecological events (every 20-40 min) provide +5-8 BEI each when handled successfully. 5-8 successful responses = BEI 95+. The push from 70 to 95 is an intense, focused challenge — not a slow grind.

### Module Construction Time

**1 module = 120s seed incubation + 48 min growth + 30 min chimera deployment + configuration.**
**Total: ~90 min per module.** 7 modules = ~10.5 hours. With 2 incubators running staggered and chimera commissioning overlapping: 7 modules built in ~8 hours.

### The Omega Organism Timeline

**Requires BEI ≥90 for the Autonomous Variant.** Realistically: BEI 90 at ~hour 15-18 (after most chimeras deployed + 4-6 hours of active tuning). Autonomous Variant appears. Player collects it + gathers other 8 archived specimens (should already be in Specimen Archive from previous eras).

**Omega Organism design + evolution + validation:** ~3-4 hours. Growth in Genesis Transcription Array: 48 min. Deployment: immediate.

**Total Omega timeline: hours 15-24.** Fits within 20-28 hour era.

### Power (Arcology Core)

At BEI 75: energy-neutral. At BEI 100: ~15 MW surplus. Player's external factory still relies on inherited generators (~46 MW). The Arcology Core is a capstone achievement, not a required power source. Post-endgame: additional arcology modules could replace external generators entirely.

## 13. IMPLEMENTATION NOTES

**Module Architecture Seeds:** Large-scale version of Era 8's living architecture seeds. The incubator recipe produces one Module Architecture Seed that, when placed, triggers a 48-minute control.lua growth timer. During growth, the module entity transitions through 4 visual stages (foundation → walls → ceiling → integration). On completion: spawns all component entities (walls, ceiling, floor, conduits, mesh, filtration) within the 12×12 footprint. Chimeras are deployed manually after growth.

**Chimera organisms:** Each XB-type chimera is a unique entity prototype. Designed via Genome Authoring Platform (specific recipe per chimera type, consuming specific cassette combinations). Evolved via Evolution Engine for fitness. Validated via Validation Chamber. Deployed by placing the validated chimera item within a module zone (control.lua checks module type — XB-07 can only be placed in Habitat modules, etc.).

**Biosphere Equilibrium Index:** Tracked by control.lua. BEI calculated every 60 seconds from: number of deployed chimeras × chimera interaction quality (mutualistic = +, competitive = -, neutral = 0) × zone health average × lymphatic efficiency × nutrient flow stability. Formula weights chimera interactions most heavily — the ecosystem's cooperation is the primary driver.

**BEI-triggered events:**
- BEI ≥90: Autonomous Variant spawns in Ecology Monitor module (one-time).
- BEI ≥95: De Novo Genome item spawns in Ecology Monitor module (one-time).
- BEI 100: Endgame sequence triggers (camera zoom, achievement, final text).

**Omega Organism (XR-00):** Genesis Transcription Array is a 4×4 assembling-machine. Recipe: 9 archived specimens (specific item per era) + Omega Genome Blueprint + Omega Gene Cassette Set. 48-minute recipe time (2880 seconds). Output: Omega Organism XR-00 (unique entity). When placed, control.lua establishes Bio Web relationships with all deployed chimeras. All competitive → neutral, all mutualistic +10%, all commensal → mutualistic.

**Arcology Core:** `electric-energy-interface`. Power output scales with BEI via control.lua: `energy_production = BEI_surplus_formula`. At BEI 75: 0 kW output. At BEI 100: 15 MW output. Linear interpolation between.

**Endgame camera:** Factorio 2.0 (Space Age) has cutscene support. Trigger cutscene on final technology research completion. Zoom from player position to island overview. Display achievement stats. Return control to player. The game continues — save remains playable.

**Ecological tuning tools:** Implemented as items consumed by the Ecological Intervention Console (assembling-machine). Each tool item → specific recipe affecting a target zone. Effects applied via control.lua modifying chimera/zone parameters. Delayed propagation: changes take effect 5 min after application (control.lua timer).

**Module blueprints:** Items crafted at the Genome Authoring Platform. Each blueprint specifies the module type and required components. The blueprint is consumed by the Living Architecture Seed Incubator alongside component items to produce the Module Architecture Seed.
